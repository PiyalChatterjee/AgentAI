## How to Use

### Planning
- Start with **ROADMAP_AI_AGENTIC.md** to understand the overall learning arc
- Check **COURSE_DAILY_SCHEDULE.md** daily to stay on track

### Progress Tracking
- Update **DAILY_CHECKIN_TEMPLATE.md** at the end of each day with:
  - What you built
  - Assessment metrics (completion, correctness, reliability)
  - Technical depth evaluations
  - Blockers and solutions
  - Reflections and next-day adjustments

### Day-Specific Learning
- For detailed notes, code, and activities, see the corresponding day folder:
  - `day_01/` → Day 1 OpenAI API & System Prompts
  - `day_02/` → (Coming soon)
  - etc.
